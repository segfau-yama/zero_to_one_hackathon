## テーマ

## 解決

# コンポーネントの分割
## FooterComponent
- カメラ、マップ、ポイント画面に遷移するボタンを表示する。

以下のコンポーネントはReactRouterで表示します。
## CameraComponent
- カメラで除雪箇所の写真を取るコンポーネント
- 撮影ボタンとカメラ画面で構成される
- 撮影後はモーダルでコメントとユーザー名を入力し、以下の形式でローカルストレージに保存する(SnowRemovalData)
  - 画像(png)、コメント、ユーザー名、座標情報
  - 
## MapComponent
- SnowRemovalDataの座標情報を参考に、leafletのマップにピンを表示する
- マップピンをクリックすると画像、コメント、ユーザー名と除雪完了ボタンが含まれたモーダルを表示する
- 除雪完了を押すとピンが削除される
- 
## PointComponent
- 擬似的にポイントを表示する画面。
- 取得したポイント以外に表示はしない。
